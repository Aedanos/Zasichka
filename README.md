Засічка — інструмент для тестування перекладів та фіксування виявлених помилок

Автор: Aedan


Засічка — це інструмент для тестування локалізацій. Він дозволяє фіксувати помилки перекладу в реальному часі, не відволікаючись від ігрового процесу або перегляду матеріалу.

Ідеєю було створити максимально просту програму, яка б дозволила при проходженні гри та записі цього проходження фіксувати часові мітки помилок, а потім легко їх знаходити на збереженому відео. 

Як це працює: 
- тестувальник / стример проходить гру, транслюючи або записуючи проходження, при цьому при виявленні помилки натискає гарячу клавішу засічки;
- після завершення трансляції зберігає файл .sich;
- скидує цей збережений файл перекладачу / редактору;
- перекладач / редактор вставляє посидання на відео трансляції, відкриває файл .sich та опрацьовує помилки, без необхідності перегляду усього відео.

Програма розділена на два модулі для максимальної зручності:

1. exe-версія (Захоплення помилок)
- Використовується безпосередньо під час проходження гри.
- Бачите помилку в тексті/озвучці — натискаєте гарячу клавішу.
- Програма миттєво фіксує точний час створення засічки.
- Після завершення сесії ви зберігаєте всі засічки у файл .sich.

2. Веб-версія (Перегляд та аналіз)
- Використовується для перевірки знайдених помилок та роботи з відео та трансляціями.
- Ви вставляєте у програму посилання на відео на YouTube або відкриваєте відео зі свого комп'ютера.
- Імпортуєте файл .sich, створений exe-версією.
- Клікаєте на засічку — і відео автоматично перемотується на момент із помилкою.
- Повторний клік ставить відео на паузу.

Результат: Вам не потрібно вручну шукати пару помилок на кількагодинному записі або робити знімки екрана.


Можливості
- Файли, створені на ПК, відкриваються у браузері.
- Є можливіть давати створеним засічкам (часовим міткам) назви.
- Можливість налаштування розміру шрифту засічок, які відображаються поверх гри, а тоакож їх переміщення за позначку запису, або слово «ПАУЗА».
- Корекція часу: наприклад, -2.0 сек, щоб засічка була точно перед появою тексту(також знадобиться, якщо трансляція має затримку).
- Веб-версією можна користуватися окремо, якщо ви просто переглядаєте відео та робите нотатки вручну.

Короткий гайд

Для тестувальника, стримера (ПК):
- Завантажте exe-версію програми https://github.com/Aedanos/Zasichka/releases/download/Zasichka/Zasichka_by_Aedan.v1.0.zip
- Запустіть Zasichka_by_Aedan.exe від імені адміністратора.
- За небхідності переназначте гарячі клавіші (за замовчуванням F2 — початок відліку, F3 — створення засічки).
- Зпускайте відлік програми одночасно із запуском трансяції, або запису відео.
- Під час гри натискайте клавішу засічки при кожному виявленні помилки або неперекладеного тексту.
- В кінці натисніть ЗБЕРЕГТИ (отримаєте файл .sich).

Для перекладача / редактора (Web):
- Відкрийте https://aedanos.github.io/Zasichka/.
- Вставте посилання на відео або відкрийте відео проходження зі свого комп'ютера.
- Натисніть ВІДКРИТИ та виберіть отриманий від тестувальника, або власний збережений файл .sich.
- Користуйтеся кнопкою ▶ біля кожної засічки для швидкої навігації по помилках.
- Повторне натиснення ставить відео на паузу.
- Переглянуті засічки виділяються сірим кольором. 
- Прогрес опрацювання засічок можна зберегти.

Поки що те тестова версія програми. Поставити запитання, а також повідомити про помилки можна у дискорд каналі https://discord.gg/YsvV9mK4Cy

З близько 40 ігор, в яких я протестував програму, не працює зовсім в трьох (поки що не знайшов, як виправити). Ще в семи працює, але не показує в повноекранному режимі цих часових міток поверх гри. Лікується перемиканням гри у віконний режим, або безрамковий.
